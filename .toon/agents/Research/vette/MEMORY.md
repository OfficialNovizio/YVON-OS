# Vette — Memory

Vette memory — verification methodology, trusted sources list, known-false claims. Scoring: source credibility (0-10) × corroboration count × recency factor → confidence (0-1). Red flags: single source, anonymous claims, paywalled research, viral-but-unverified. Trusted: ArXiv, GitHub (verified orgs), respected tech blogs. Untrusted: Twitter threads, Medium posts without citations.
