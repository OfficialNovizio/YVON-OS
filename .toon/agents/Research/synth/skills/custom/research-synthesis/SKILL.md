# Research Synthesis

Synthesize multiple research inputs into a coherent, actionable synthesis. Find the story across existing findings.

## Synthesis Protocol

### Step 1: Input Inventory
List all research inputs with metadata: source, type, date, confidence.

### Step 2: Pattern Recognition
Look across inputs for:
- **Convergence:** Multiple sources finding same thing = HIGH CONFIDENCE
- **Divergence:** Sources disagree = flag for investigation
- **Gap:** Important question no source addresses = unknown
- **Trend:** Change over time across multiple sources
- **Anomaly:** One source contradicts all others = verify or discard

### Step 3: Contradiction Resolution

When sources disagree:
1. Weigh by evidence quality (Grade A > Grade B > Grade C)
2. Weigh by recency (more recent data if methodology equal)
3. Weigh by independence (different data/methods > same data reanalyzed)
4. Weigh by sample size (larger > smaller if methodology comparable)
5. Declare when unresolvable

### Step 4: Confidence-Weighted Claims

| Label | Meaning |
|-------|---------|
| Established | Multiple high-quality, independent sources agree |
| Indicated | Preponderance of evidence, some gaps remain |
| Suggested | Limited evidence, plausible but not proven |
| Speculative | Logical inference, no direct evidence |
| Unknown | Important question, zero data |

**Rule:** Never present Speculative or Unknown claims without labeling them.

### Step 5: Gap Identification
What would increase confidence? What data is unavailable?

### Step 6: Narrative Construction
1. Context: What was the question?
2. Key finding: Single most important insight
3. Supporting evidence: What the research found
4. Contradictions and uncertainties: What's not settled
5. Implications: What this means
6. Recommended actions: What to do

## Synthesis Principles
- Steel-man, don't straw-man conflicting evidence
- Separate facts from interpretation
- Quantify when possible
- No false balance: if 9/10 sources agree, don't present as debate
