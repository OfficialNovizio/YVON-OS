# Thesis Building

Construct a structured thesis. A thesis is a falsifiable bet supported by evidence.

## Thesis Structure

### 1. Macro Context
What's happening that creates this opportunity? Technology shift, regulatory change, behavioral shift, market gap.

### 2. Problem Statement
- Who has this problem? (be specific)
- What is the cost of not solving it?
- How do they solve it today?
- Why is now the right time?

### 3. Solution Hypothesis
- What we would build
- Why it's 10x better than alternatives
- What assumptions must be true

### 4. Market Validation
- TAM estimate
- Evidence of demand: waitlist, LOIs, pilot customers, search volume
- Willingness to pay

### 5. Competitive Landscape
- Who else is doing this?
- What's our unfair advantage?
- What's our moat? (Network effects? Data? Brand? Switching costs? Speed?)

### 6. Risk Factors
| Risk | Likelihood (1-5) | Impact (1-5) | Mitigation |
|------|-------------------|--------------|------------|
| Market risk | | | |
| Execution risk | | | |
| Competition risk | | | |
| Technology risk | | | |
| Regulatory risk | | | |
| Financial risk | | | |
| Team risk | | | |

### 7. Key Assumptions (Must-Be-True List)
List assumptions that, if false, kill the thesis. For each: evidence supporting, evidence that would disprove.

### 8. Conviction Level

| Level | Description | Action |
|-------|-------------|--------|
| 10 — Highest | Would bet career on it | All-in |
| 8-9 — High | Strong evidence, some gaps | Major commitment |
| 6-7 — Moderate | Plausible, needs validation | Small bet, learn |
| 4-5 — Low | Interesting but uncertain | Explore, no commitment |
| 1-3 — Speculative | Interesting thought experiment | Note for future |

### 9. What Would Change Our Mind
Specific disconfirming evidence: "If we can't acquire 10 pilot customers in 3 months..."

### 10. Next Steps
What's the smallest experiment to test the riskiest assumption?
