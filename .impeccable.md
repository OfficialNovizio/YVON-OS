# Design Context

## Users
**Primary user:** Stark — sole founder and operator of YVON, managing two ventures (Novizio fashion brand + Hourbour fintech SaaS). Uses the dashboard daily as a CEO command center.

**Jobs to be done:**
- Get a daily briefing on venture health before making decisions
- Review and approve decisions flagged by AI agents
- Monitor anomalies, market intelligence, and team activity
- Switch between ventures (Novizo / Hourbour) with different agent teams per venture

**Context:** Solo operator running multiple brands. Needs clarity, speed, and confidence — not exploration.

## Brand Personality
**3 words:** Commanding, lucid, alive

**Voice:** Direct and professional. No corporate polish, no startup playfulness. The interface should feel like a mission control terminal run by an elite operator.

**Emotions:** Confidence, clarity, control. The CEO should feel like they have a pulse on everything happening across ventures.

## Aesthetic Direction
**Visual tone:** Dark, premium, green-accented. Dense but breathing. Information-dense layout with clear hierarchy — no wasted pixels.

**References:**
- Bloomberg terminal meets Apple design system — functional density at premium quality
- The green wave hero as a signature moment — organic fluid shapes against rigid grid

**Anti-references:**
- Generic SaaS dashboard (Linear/Vercel clone vibes)
- Playful startup aesthetics
- Glassmorphism, sparklines, card nesting

**Theme:** Dark mode only — deep dark base with luminous green (#22C55E) as the sole accent color.

## Design Principles
1. **Command density** — show everything on one screen without scrolling if possible. Information is the design.
2. **Green is the signal** — every interaction point, every status indicator, every CTA uses green. No secondary accent colors.
3. **Hero is the moment** — the wave hero image is the emotional anchor. Everything else is subordinate.
4. **Monospace for data, serif for voice** — `DM Mono` for metrics, data, labels. `Playfair Display` for headings, briefings, and voice.
5. **Zero decoration** — every element exists to convey data or enable action. No icons-as-decoration, no decorative borders without function.
